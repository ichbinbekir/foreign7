# Foreign7 - Language Learning App Task List

## ✅ Completed Features
- [x] **Project Setup:** Go, Ollama SDK, and Bubble Tea integration.
- [x] **Architecture (Refactoring):** Stack-based navigation with `tearouter` and modular structure under `internal/model`.
- [x] **Centralized Data Management:** Multi-list and active list management with `.txt` based files via `internal/data`.
- [x] **Mode 1 (Meaning Guessing):** Writing the Turkish meaning of the word and LLM feedback.
- [x] **Mode 2 (Sentence Construction):** Using the word in a sentence, grammar and naturalness analysis with LLM.
- [x] **Library System:**
    - [x] Dynamic list scanning and activation by checking (Multi-select).
    - [x] Creating a new list (`CreateListModel`).
    - [x] Global search (Scanning all active lists and showing origin/file name).
    - [x] Adding words (Ollama validation + Duplication check + Suggestions).
    - [x] **Centralized Storage:** Lists are now kept in the user cache directory (`~/.cache/foreign7`).
    - [x] **Export:** Exporting lists by pressing 'e' key over them (Backup).
    - [x] **Import:** Including `.txt` files from the computer into the library.
    - [x] **Delete List:** Deleting via 'x' key over lists. Forces creating a new list when the last one is deleted.
    - [x] **Delete Word:** Deleting individual words from the library (via 'x' key on search results).
- [x] **Technical Improvements:**
    - [x] **Advanced Prompting:** Dynamic management of system instructions via `assets/prompts.json`.
    - [x] **Build System:** Automatic copying of assets and compilation under `bin/` with Makefile.
    - [x] **Multi-language Support:** TR/EN interface options with JSON files under `assets/lang`.
    - [x] **Data Layer Refactoring:** Separation of code under `internal/data` into modular files (`config`, `prompts`, `wordlists`).
    - [x] **Routing Evolution:** Upgraded `tearouter` to v1.1.0 and implemented hierarchical subrouting for better organization.

## 🛠 Current Architecture Notes
- **Main File:** `cmd/foreign7/main.go` (Router definitions and initialization)
- **Data Layer:** `internal/data/*.go` (Separated into Config, Prompts, Wordlists, and Store)
- **Data Path:** `~/.cache/foreign7/*.txt` (Direct cache usage)
- **Models:** `internal/model/*.go` (Each screen is a model: menu, test, sentence_mode, manager, list_select, create_list, import_list, settings)
- **Model:** `translategemma:latest` (Ollama)
- **Prompts:** `assets/prompts.json` (JSON based template system)
- **Language Files:** `assets/lang/*.json` (Interface translations)
